# AI功能模块

该目录包含MindNote的AI内容分析集成功能。

## 目录结构

```
src/lib/ai/
├── services/          # AI服务实现
├── providers/         # AI提供商集成
└── config/           # AI配置管理
```

## 核心功能

- 智能文本分析
- 自动内容分类
- 标签提取
- 向量化存储
- 多模型AI集成

## 技术栈

- Next.js 15 + React 19 + TypeScript
- Vercel AI SDK
- OpenAI API, Anthropic API
- PostgreSQL + pgvector
