## 📝 使用说明

### 查看会话记录
1. 浏览上方的会话列表
2. 点击感兴趣的会话链接
3. 查看详细的分析内容、决策过程和行动指南

### 发起新的协作会话
使用以下命令启动新的AI协作：
```bash
# 启动特定范式的协作
/ai.collab start [范式] "主题描述"

# 示例
/ai.collab start visual "设计用户界面原型"
/ai.collab start creative "产品功能头脑风暴"
/ai.collab start critical "技术方案风险评估"
```

### 保存协作成果
完成协作后，记得保存成果：
```bash
/ai.collab save
```
